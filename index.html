Matthew Neumeyer ePortfolio

Objective: This is an ePortfolio to document and display academic and independent projects.

Academic Projects

  • Insurance Website Assignment: For this assignment, a health insurance website needed to be developed. The website needed to be able to process a person's information that they input.
  • Rate Measurement Assignment: For this assignment, a program that analyzed birth and death rate populations of two types of animals. The birth and death rates were measured over time, and a graph that displayed the number of the two animals in each population was made. The number of animals in each population were calculated from the birth and death rates of the two animal populations.
  • Simple Data Analysis Assignment: For this assignment, a program that read and analyzed data from a dataset was created. Simple data analysis was performed, rows and columns were viewed and edited, and histograms were created. The code was created in Python.
  • Data Association Analysis Assignment: For this assignment, a program that analyzed the association between two data variables was created. Different variables were analyzed from a dataset, and graphs that show the association and regression between two variables were made. The program was created in Python.
  • Database Assignment: For this assignment, a database was created in Python using the sqlite package. The data in the database was viewed using SQL commands in Python.

Independent Projects

  • SQL Query Project: COVID-19 data was queryed and analyzed using SQL. Views were created in SQL that would be used to visualize the data in Tableau.
  • SQL Data Cleaning Project: Nashville housing data was queryed and analyzed in SQL. Data was cleaned in SQL to make the data more usable for analyses.
  • Tableau COVID-19 Data Visualization Project: COVID-19 data was visualized in Tableau. Queries from SQL was used to create Excel spreadsheets, which were then imported into Tableau Public for data visualization.
  • Tableau Monkeypox Data Visualization Project: Monkeypox data was visualized in Tableau. A CSV file was converted to an Excel file, which was then imported into Tableau Public for data visualization.
